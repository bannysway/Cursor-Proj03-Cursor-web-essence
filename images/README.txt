# 图标文件说明

请在此目录中放置以下图标文件：

1. icon16.png (16x16像素)
2. icon48.png (48x48像素)
3. icon128.png (128x128像素)

您可以使用任何图像编辑工具创建这些图标，或者从网上下载适合的图标并重命名。

图标应该与扩展的功能相关，例如可以使用放大镜、文档或摘要相关的图标。

## 临时图标解决方案

如果您暂时没有合适的图标，可以使用以下在线工具生成简单的图标：

1. 访问 https://favicon.io/ 或类似的图标生成网站
2. 创建一个简单的文本图标，例如使用字母"C"或"S"
3. 下载生成的图标并重命名为上述文件名
4. 放置在此目录中

请确保图标文件的格式为PNG，并且大小符合要求。
